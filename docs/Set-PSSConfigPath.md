# Set-PSSConfigPath

## SYNOPSIS
Sets the path to the active PSSimpleConfig configuration file.

## SYNTAX

```
Set-PSSConfigPath [-Path] <FileInfo> [<CommonParameters>]
```

## DESCRIPTION
Sets the path to the active PSSimpleConfig configuration file.
The path
must be to an existing file.
